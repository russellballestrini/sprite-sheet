Sprite Atlas Example

Managing multiple sprites from a single texture atlas

Select Character:

How It Works

  • Auto-Detection: Frame dimensions are automatically calculated from image size
  • Sprite Regions: Define multiple sprites in one atlas using x, y coordinates
  • Easy Management: Name and switch between sprites easily
  • Efficient: Single image load for all sprites

Example: Auto-Detecting Dimensions

// Auto-detect frame size from a 4x3 sprite sheet (12 frames) const sprite = new SpriteSheet({ canvas: '#myCanvas', image: 'characters.png', frames: 12, // Total frames rows: 4, // 4 rows columns: 3 // 3 columns // frameWidth and frameHeight calculated automatically! }); await sprite.ready(); console.log(sprite.getInfo()); // See detected dimensions sprite.play();

Example: Using Sprite Atlas (Multiple Characters on One Sheet)

import { SpriteAtlas } from './src/SpriteSheet.js'; // Create atlas from a single image containing all characters const atlas = new SpriteAtlas('#canvas', 'game-sprites.png'); await atlas.ready(); // Define different character regions in the atlas atlas .define('player', { x: 0, y: 0, // Position in atlas width: 192, height: 32, // Region size frames: 6, rows: 1 // Animation frames }) .define('enemy1', { x: 0, y: 32, // Below player width: 128, height: 32, frames: 4, rows: 1 }) .define('enemy2', { x: 0, y: 64, // Below enemy1 width: 160, height: 32, frames: 5, rows: 1 }) .define('powerup', { x: 0, y: 96, width: 128, height: 32, frames: 4, rows: 1, framerate: 150 // Slower animation }); // Play any sprite by name await atlas.play('player'); // Switch sprites document.getElementById('btn').onclick = () => { atlas.play('enemy1'); };

Example: Passing Specific Dimensions

// Method 1: Specify exact frame dimensions const sprite1 = new SpriteSheet({ canvas: '#canvas', image: 'sprite.png', frameWidth: 32, // Exact width frameHeight: 32, // Exact height frames: 8 }); // Method 2: Let it calculate from rows/columns const sprite2 = new SpriteSheet({ canvas: '#canvas', image: 'sprite.png', frames: 12, rows: 3, // Image will be divided into 3 rows columns: 4 // and 4 columns }); // Method 3: Use region for part of a larger atlas const sprite3 = new SpriteSheet({ canvas: '#canvas', image: 'atlas.png', regionX: 100, // Start at x=100 in the image regionY: 200, // Start at y=200 in the image regionWidth: 256, // Use this much width regionHeight: 64, // Use this much height frames: 8, rows: 2 });

Example: Get Image Dimensions on Load

const sprite = new SpriteSheet({ canvas: '#canvas', image: 'sprite.png', frames: 8, rows: 2, onImageLoad: (info) => { console.log('Image loaded!', info); // { // imageWidth: 256, // imageHeight: 64, // frameWidth: 32, (calculated) // frameHeight: 32, (calculated) // rows: 2, // columns: 4, (calculated) // totalFrames: 8 // } } }); await sprite.ready(); // Or get info anytime const info = sprite.getInfo(); console.log(info);
